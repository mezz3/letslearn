ตรีโกณมิติ

อัตราส่วนตรีโกณมิติบนสามเหลี่ยมมุมฉาก

กำหนดรูปสามเหลี่ยม ABC มี C เป็นมุมฉาก ความยาวด้านตรงข้ามมุม A , B และ C เท่ากับ a , b และ c หน่วย ตามลำดับ ดังรูป



เรียกอัตราส่วนต่อไปนี้ว่า อัตราส่วนตรีโกณมิติ ดังนี้



อัตราส่วนตรีโกณมิติของมุมที่ต้องจำ



สูตรปีทาโกรัสของตรีโกณ

sin 2 θ + cos 2 θ = 1
cosec 2 θ − cot 2 θ = 1
sec 2 θ − tan 2 θ = 1

สูตรโคฟังก์ชัน



สูตรผลรวมและผลต่างมุม

sin ( A + B ) = sin A cos B + sin B cos A
sin ( A − B ) = sin A cos B − sin B cos A
cos ( A + B ) = cos A cos B − sin A sin B
cos ( A − B ) = cos A cos B + sin A sin B
tan ( A + B ) = ( tan A + tan B ) / ( 1 − tan A tan B )
tan ( A − B ) = ( tan A - tan B ) / ( 1 + tan A tan B )

สูตรเปลี่ยนคูณเป็นบวก

2 sinα cosβ = sin ( α + β ) + sin ( α − β )
2 cosα sinβ = sin ( α + β ) − sin ( α − β )
2 cosα cosβ = cos ( α + β ) + cos ( α − β )
2 sinα sinβ = cos ( α − β ) − cos ( α + β )

สูตรเปลี่ยนบวกเป็นคูณ



สูตรมุมสองเท่า

sin 2x = 2 sin x cos x
cos 2x = cos 2 x - sin 2 x
cos 2x = 2 cos 2 x - 1
cos 2x = 1 - 2 sin 2 x
tan 2x = ( 2 tan x ) / ( 1 - tan 2 x )

ในรูปสามเหลี่ยม ABC ใด ๆ ถ้าความยาวด้านตรงข้ามมุม A , B และ C ยาว a , b และ c หน่วย ตามลำดับแล้ว จะได้ว่า

a 2 = b 2 + c 2 - 2bc cos A
b 2 = c 2 + a 2 - 2ca cos B
c 2 = a 2 + b 2 - 2ab cos C
